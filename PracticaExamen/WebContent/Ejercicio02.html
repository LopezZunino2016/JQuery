Crear 3 titulares de noticias, que se llamen noticia1, noticia2 y noticia3. Cuando el usuario situe el ratón encima de cada boton tiene que aparecer el contenido de la noticia, que estará dentro de una capa. El contenido de la capa (es decir el contenido de la noticia) puede tener texto y fotografías. Cuando el usuario aleje el ratón del botón la noticia (la capa) debe de desaparecer. Usuar solo una capa para mostrar las 3 noticias.
